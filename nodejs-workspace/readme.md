# OA 系统核心源代码目录

## 目录结构

src
├─api               接口定义文件
├─assets			静态资源文件
├─prototypes		原型修改脚本
├─router			VUE路由脚本
├─utils			    基础工具脚本
├─views				VUE文件
│  ├─core			核心视图文件
│  ├─customer		客户模块
│  ├─order			订单模块
│  └─system			系统模块
└─vuex-store		VUEX状态存储文件

## 脚本


### 开发模式

> npm run dev

### 构建模式

> npm run build
