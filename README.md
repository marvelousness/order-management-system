# 订单管理系统

> 该系统主要用于毕业论文，仅供学习。旨在为电商公司提供日常的客户和订单信息录入、订单分配以及订单报表查询的基本功能。

它能减轻电商公司业务员的工作压力，比较系统的对客户信息、订单信息进行管理，同时，可以减少劳动力的使用，加快查询速度、加强管理，使各项管理更加规范化。
